Auto Employee Creation From Users

This module helps the user to create an employee at the time of creating a user or from existing user, on the user page.

Features

Auto employee creation

Automatically create employee from the user when create a new user.

Less time consuming

Reducing the amount of time needed to create user and employee separately.
Demo Request Video Tutorial

Option to create employee

At the time of creating users can create the same name as employee with or without job position.


Employee page

The new employee will be added, with the same name, and job position into employee page.


Uncheck create employee checkbox

Uncheck the create employee checkbox to inactive the employee.


Archived Employee in the filtered page

After filtering, you can see all the archived employees.
